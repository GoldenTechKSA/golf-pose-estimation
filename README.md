# SwingLens

AI-powered golf swing analysis platform. Upload a video of your swing, get back an annotated
video with pose skeleton overlay, automatic swing-phase detection, biomechanical metrics, and
AI-generated coaching feedback.

> **Status:** under active development. This repo began as a CSE 455 computer vision class
> project (preserved in [`legacy/`](legacy/)) and is being rebuilt into a full-stack platform:
> Next.js frontend + FastAPI ML backend + Celery/Redis async processing pipeline.

Full documentation, architecture diagram, and setup instructions land with the infrastructure PR.
